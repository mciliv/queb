# SmartCLI\n\nSmart terminal wrapper using Gemini CLI to classify input:\n- Shell commands\n- LLM queries (gemini)\n- Web searches (ddgr)\n\n## Setup\n\n1. Install Gemini CLI:\n   ```bash\n   npm install -g @google/gemini-cli\n   gemini --setup  # or set GEMINI_API_KEY\n   ```\n   See: https://github.com/google-gemini/gemini-cli\n\n2. Install ddgr:\n   ```bash\n   pip install ddgr\n   ```\n\n## Run\n\n```bash\nchmod +x smartcli.py\n./smartcli.py\n```\n\n## Features\n- Interactive REPL\n- Auto-classifies input\n- Handles errors gracefully\n\nExamples:\n> ls -la  → shell\n> What is AI? → gemini\n> python tutorial → ddgr
